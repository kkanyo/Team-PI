# PlayIdea
>### 스마일게이트멤버십(SGM) 10기 PlayIdea
>
>모바일 RPG, 라이트 로그라이크 게임 ‘마을은 내일 구하자’
---

#### 주인공의 인내심과 성실함을 관리하며 타이밍에 맞게 무기를 던져 괴물을 처치하자!

1. 게임장르: 2D 로그라이크
2. 사용 플랫폼: Android
3. 제작 인원: 3인 (개발 2인, 디자인 1인)
4. 제작 기간: 5월 ~ 12월
5. 사용 엔진: Unity
6. 담당 부분: UI 및 전투 시스템 구현
